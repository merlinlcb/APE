How do i use A.P.E. ?

SMUI Additional Page Editor(A.P.E. for short) is easy to use, the fields available in it are identical in purpose to the ones provided in streamdeck-linux-gui
the only thing you dont have to define that you MUST for this application is a page number, you can defined any pages beyond 10
this also expects that you import your existing .json config, where that is stored on your device will depend on your operating system.
this has only currently been tested using the 15 key model of streamdeck, please let me know if it works with the larger ones.

PAGES DEFINED USING THIS UI CANNOT BE VIEWED IN THE ORIGINAL SOFTWARE! YOU WILL NEED TO MANUALLY EDIT THE SwitchPage option to get to your generated pages onto an existing page (0-9) as the original software will not allow you to define anything over 10
